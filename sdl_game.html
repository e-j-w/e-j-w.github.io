Untitled RPG

A solo-developed 2D Role-Playing Game (RPG). Developed as a hobby for my own personal gratification, may or may not be released at some point.

Expected Features

  • Multi-platform (Linux, Windows).
  • Custom game engine based on SDL2, written in performant C99 with low hardware resource utilization.
  • Pixel art graphics which rival the Game Boy Advance, but at slightly higher resolution.
  • Gamepad support.
  • Hours of fun!

Development status

Basic engine code (rendering, input, audio, data formats, etc.) is finished. Now implementing gameplay systems (slowly). The first 10-15 minutes of the game are playable. Probably years from completion.
